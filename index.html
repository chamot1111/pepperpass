Pepperpass

simple apps to salt keyword

View on GitHub Download .zip Download .tar.gz

Use always the same password for all the website is very dangerous. It means that the security of all those websites is based on the weakest of all the websites.

This is a program to create and recreate always the same password for each websites. Just remember a password and make it change based on the website name.
